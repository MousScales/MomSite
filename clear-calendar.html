Clear Google Calendar Events

Note: To clear Google Calendar events, you'll need to manually delete them from your Google Calendar.

How to clear Google Calendar events:

  1. Go to Google Calendar
  2. Find the calendar where booking events were created
  3. Select all events related to Maya African Hair Braiding
  4. Delete them (or you can delete the entire calendar if it was created specifically for bookings)

What has been cleared:

  • ✅ All Firebase Firestore bookings (84 documents deleted)
  • ✅ All browser localStorage booking data
  • ✅ All sessionStorage booking data
